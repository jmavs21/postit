### API

## Technologies used
- Kotlin
- Spring Boot
  - Spring Web
  - Spring Data JPA
  - Spring Security with JWT
  - Spring Validation
- H2 Database
- Java JWT
- Swagger

## Installation
- Go into project
```sh
cd /api
```
- Build
```sh
./gradlew build
```
- Run
```sh
./gradlew bootRun
```

## Usage
By using [Swagger](http://localhost:4000/swagger-ui.html#/)
