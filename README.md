# Posts

A posts web application.

## Features

- Register a new user
- Login and logout with JWT
- Create, update and delete a post
- Voting on posts
- Search posts
- Follow users
- Pagination
- Light and dark modes

## Technologies used

### Website

- TypeScript
- React
- Chakra UI
- React-router-dom

### Server

- Kotlin
- Spring Boot
  - Spring Web
  - Spring Data JPA
  - Srping Security with JWT
  - Spring Validation
- H2 Database
